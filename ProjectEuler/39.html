Project Euler Problem 39

Description

If p is the perimeter of a right angle triangle with integral length sides, {a,b,c}, there are exactly three solutions for p = 120.

{20,48,52}, {24,45,51}, {30,40,50}

For which value of p <= 1000, is the number of solutions maximised?

Solution

求某个数p(p<=1000)使得以p为周长的直角三角形个数最多。

我们可以定义一个1000大小的数组p,p[n]代表周长为n的三角形的个数，枚举直角三角形的两边，通过这两边求出斜边，从而求出相应的周长n,将p[n]加上1，最终遍历p数组，其最大值的索引即为答案。

在枚举直角三解形的两直角边时，我们另a<=b，则根据直角三角形直角边小于斜边的定义，当周长最大为1000时，a最大只能到333，因为当另b=a时，有1000-2*a=334，这是斜边所能取的最小值，a再大就会导致斜边小于a，同理b最大只能到499，因为当a取小值1时，1000-a-b=500，b再大会导致斜边小于b，同时在内层循环一但发现1000-a-b<=b则可以立即跳出内层循环。
