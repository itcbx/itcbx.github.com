Project Euler Problem 19

Description

You are given the following information, but you may prefer to do some research for yourself.

  • 1 Jan 1900 was a Monday.
  • Thirty days has September,
    • April, June and November.
    • All the rest have thirty-one,
    • Saving February alone,
    • Which has twenty-eight, rain or shine.
    • And on leap years, twenty-nine.
  • A leap year occurs on any year evenly divisible by 4, but not on a century unless it is divisible by 400.

How many Sundays fell on the first of the month during the twentieth century (1 Jan 1901 to 31 Dec 2000)?

Solution

已知1900年1月1日是星期一，求从1901年1月1日到2000年12月31日这段时间，星期天落在每个月的第一天的次数。

因为1900年1月1日是星期一，那么从1899年12月31号往后的第n天，将n对7求余，当余数为0，代表这第n天为星期天，余数为1到6则分别代表这第n天为星期一到星期六。在计算某天为从1899年12月31号开始的第几天时，要注意到闰年的变化。

Code

#!/usr/bin/python3
# -*- coding: utf-8 -*-

import math
import re
from timeit import Timer

table = [0, 
         0, 31, 28, 31,
         30, 31, 30, 31,
         31, 30, 31, 30]

def count_sunday_1st_of_month_1901_2000():
    res = 0
    t = 1
    for year in range(1900, 2001):
        if ((year%100 != 0 and year%4 == 0) or 
            (year%100 == 0 and year%400 == 0)):
            table[3] = 29
        else:
            table[3] = 28
        for i in range(1, 13):
            t += table[i]
            if year > 1900 and t % 7 == 0:
                res += 1
        t += 31
    return res

if __name__ == "__main__":
    print(count_sunday_1st_of_month_1901_2000())
