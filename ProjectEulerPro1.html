Project Euler Problem 1

Description

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000.

Solution

A simple way is to go through all numbers form 1 to 999 and test whether they are divisible by 3 or 5.

But the solution is too slow to run if we change the number from one thousand to one billion. To get a more efficient solution, you could calculate the sum of the numbers less than 1000 that are divisible by 3, plus the sum of the numbers less than 1000 that are divisible by 5, and subtract the sum of the numbers less than 1000 that are divisible by 15 as you have summed numbers divisible by 15 twice. The formula is showed below:

\[ sum(n) = \sum_{i=1}^{\lfloor \frac{n}{3} \rfloor} 3i + \sum_{i=1}^{\lfloor \frac{n}{5} \rfloor} 5i - \sum_{i=1}^{\lfloor \frac{n}{15} \rfloor} 15i \]

At this problem, the n is 999. As you can see, the formula is likely to the inclusion-exclusion principle for two set.

Code

#include <iostream>
using namespace std;
int sum(int n) {
    return (1+n)*n*0.5;
}
int main() {
    cout << 3*sum(999/3) + 5*sum(999/5) - 15*sum(999/15) << endl;
    return 0;
}
